FaceSecure Pay is a high-integrity financial transaction simulator. This project bridges the gap between Computer Vision (AI) and Distributed Ledger Technology (Blockchain) to build resilient software architectures for modern financial applications.

---

### System Architecture
The application operates on a triple-layer framework designed for high data integrity and identity verification:

| Layer | Technology | Function |
| :--- | :--- | :--- |
| **Identity Management** | OpenCV / Biometrics | Real-time facial feature extraction and authentication. |
| **Data Integrity** | SHA-256 Blockchain | Recursive cryptographic linking of every transaction record. |
| **Persistent Storage** | SQLite3 / Hashing | Secure storage with an integrated "Tamper Detection" engine. |

---

### Engineering Implementation

#### Blockchain Integration for Data Integrity
The core ledger utilizes a recursive hashing algorithm where each transaction block contains the hash of the preceding block. This ensures an immutable record of data, where any unauthorized database modification is instantly flagged by the system.

#### Computer Vision Authentication
Integrated vision protocols facilitate seamless "Register New Face" actions. This simulates a production-ready FaceID payment gateway, providing a robust layer of biometric authentication within a Software Engineering framework.

#### Enterprise Analytics & Telemetry
A minimalist dashboard providing real-time system telemetry:
* **Protocol Health:** Continuous monitoring of system-wide status.
* **Identity Analytics:** Real-time tracking and indexing of registered users.
* **Ledger Synchronization:** Automated validation of the blockchain chain on every page request.
